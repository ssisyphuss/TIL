### 예외처리

#### 구문오류와 예외



#### 예외 처리 방법

- if A else B : A가 아니면 B를 해라.
- try A except B : A를 해보고 에러가 나면 B를 해라.
- try A except B else C : A를 하다가 에러가 나면 B를 하고 안 나면 C를 해라
- try A except B else C finally D : A를 하다가 에러가 나면 B를 하고 안 나면 C를 하되 모든 경우에 D는 해라.
